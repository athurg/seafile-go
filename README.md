# seafile-go
Golang版本的Seafile客户端，基于Seafile服务器提供的Web API实现。

# 当前支持的接口
- [ ] 基础接口（Basic）
  - [x] Token获取（AuthToken）
  - [x] Ping
  - [x] 认证Ping（Auth Ping）
- [x] 账户（Account）
  - [x] 获取账户信息
  - [x] 获取服务器信息
- [ ] 资料库
  - [x] 获取资料库列表
  - [x] 获取资料库上传链接
  - [x] 获取资料库更新链接
- [ ] 文件
  - [x] 上传文件
  - [x] 更新文件
  - [x] 删除文件
- [ ] 目录
  - [x] 获取目录内容
  - [x] 创建目录
